# Learn730

Free AWS Cloud Practitioner (CLF-C02) study app by Solo730 Multimedia LLC.

## What's Inside

- **Flashcards** — 134 cards covering every topic on the CLF-C02 exam
- **Glossary** — Plain English definitions for cloud terms and AWS services
- **Practice Quiz** — 91 questions across all 4 exam domains with full explanations
- **Exam Info** — Test-taking tips, domain weights, and step-by-step registration guide

## How to Use

No login. No paywall. No API. Just open it and study.

Live at: **solo730.github.io/learn730**
