# ⚡ Learn730

A free, offline-first mobile study app for the **AWS Certified Cloud Practitioner (CLF-C02)** exam. Built by Solo730 Multimedia LLC for anyone looking to break into cloud — no background required.

No login. No paywall. No internet required. Just open it and study.

🌐 **Live App: [solo730.github.io/learn730](https://solo730.github.io/learn730)**

---

## 📱 Features

### 📚 Flashcards (134 cards)
- Covers all CLF-C02 exam domains and major AWS services
- Tap to flip between question and answer
- Filter by topic section
- Shuffle mode for randomized practice
- Progress bar tracking

### 🎯 Practice Quiz (92 questions)
- Multiple choice with instant feedback
- Color-coded correct / incorrect answers
- Full explanation after every question — correct answer first, then why each distractor is wrong
- Score screen with personalized feedback and XP rank
- Best score saved locally on your device
- Questions randomized every session

### 📖 Glossary (36 terms)
- Plain English definitions for cloud terms and AWS services
- Tap any term to expand its definition

### 🗂️ Exam Info
- Domain weights and topic breakdown
- Test-taking tips
- Step-by-step exam registration guide

## 🚀 How to Use

**Live (Recommended)**
1. Visit [solo730.github.io/learn730](https://solo730.github.io/learn730) on any device
2. On mobile, tap **Add to Home Screen** for an app-like experience
3. Works offline after first load

**Download**
1. Download `index.html`
2. Open in Chrome (Android), Safari (iOS), or any desktop browser
3. No install, no dependencies — works immediately

**Study Strategy**
1. **Flashcards first** — go through all 134 cards at least twice
2. **Use the Glossary** — look up any unfamiliar terms
3. **Take the Quiz** — aim for 70%+ (that's the real exam passing threshold)
4. **Repeat** — shuffle the cards and retake until you're consistent
